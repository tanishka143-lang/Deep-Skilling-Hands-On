To understand and perform basic Git operations.

\---



\##  Commands Used



\### Configure Git

```

git config --global user.name "Your Name"

git config --global user.email "your-email@example.com"

```



\### Initialize Repo

```

git init

```



\### Add File

```

git add welcome.txt

```



\### Commit

```

git commit -m "Initial commit"

```
