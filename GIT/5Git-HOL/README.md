# 5Git-HOL - Clean Up and Push to Remote

## Objectives

- Explain clean up in Git
- Push changes to remote repository

---

Git clean-up refers to maintaining a clean working directory, removing unnecessary branches, and syncing with remote repositories.
Git push is used to upload local commits to a remote repository like GitHub.

---

## Steps Performed

### 1. Verified clean state

git status

### 2. Listed all branches

git branch -a

### 3. Pulled latest changes

git pull origin main

### 4. Pushed changes to remote

git push origin main

---

## Outcome

- Verified repository is clean
- Synced local and remote repository
- Successfully pushed changes to GitHub
