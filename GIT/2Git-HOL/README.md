# Git Ignore Hands-On

## Objective

- Understand git ignore
- Learn how to ignore unwanted files and folders

## Tasks Performed

- Created a `.log` file
- Created a `log` folder
- Configured `.gitignore` to ignore:
  - \*.log files
  - log folder

## Commands Used

git init  
git status  
git add  
git commit  
git push

## Outcome

Successfully ignored unwanted files using `.gitignore`.
