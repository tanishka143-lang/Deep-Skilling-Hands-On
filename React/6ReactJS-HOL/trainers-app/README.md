## Objective

- Implement navigation using React Router
- Create routes in React application
- Pass parameters using URL

- React JS
- React Router DOM
- VS Code
- Node.js

## Implementation Steps

1. Created React app using create-react-app
2. Installed react-router-dom
3. Created Trainer class
4. Added mock data
5. Created components:
   - Home
   - TrainersList
   - TrainerDetails
6. Implemented navigation using Link
7. Configured routing using:
   - BrowserRouter
   - Routes
   - Route
8. Used URL parameter using useParams hook
